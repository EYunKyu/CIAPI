PartBasedCalibration(CalibrationType, Boolean, Boolean, Double[], List<CAlpsPointFeature>)

각 Step 마다 검출 된 Point 정보, Stage or Robot 위치 정보 들 을 기반으로 Stage or Robot 의 회전 중심 좌표계를 (캘리브레이션 결과) 계산 합니다 (캘리브레이션 총 11 Step)

시퀀스

  1. IsInit (true) 신호를 받게 되면 모든 정보 초기화
  2. 반복 되는 Step 마다 Point 정보, Stage or Robot 위치 현재 좌표 누적
  3. _IsCompute (true) 신호를 받게 되면 현재 까지 누적된 정보를 이용 하여 캘리브레이션 결과 계산
